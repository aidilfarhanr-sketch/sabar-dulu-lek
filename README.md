# Sabar Dulu Lek

**Sabar Dulu Lek** adalah game web santai berbasis HTML, CSS, dan JavaScript yang dibuat untuk membantu pengguna menenangkan diri saat sedang kesal, suntuk, pusing, capek emosional, atau overthinking.

Game ini menghadirkan konsep ringan, interaktif, lucu, dan menenangkan dengan karakter pendamping bernama **Mood Buddy Lek**. Game ini bukan game kompetitif berat, tetapi mini game relaksasi yang dapat dimainkan langsung melalui browser.

## Deskripsi Singkat

**Sabar Dulu Lek** dibuat sebagai game relaksasi sederhana untuk menemani pengguna ketika sedang mengalami emosi negatif seperti kesal, lelah, overthinking, atau butuh waktu untuk menenangkan diri.

Di dalam game ini, pengguna bisa memilih beberapa mode permainan seperti **Tap Emosi**, **Breath Tap**, **Pukul Bantal**, dan **Fokus 60 Detik**. Setiap mode dirancang agar pengguna dapat berinteraksi dengan cara yang ringan, menyenangkan, dan tidak membuat stres.

---

## Latar Belakang

Banyak orang sering merasa kesal, capek, pusing, atau overthinking, tetapi tidak selalu punya tempat untuk melampiaskan emosi secara aman. Dari masalah sederhana tersebut, game **Sabar Dulu Lek** dibuat sebagai media hiburan ringan yang bisa membantu pengguna mengambil jeda sejenak.

Game ini memberikan pengalaman bermain yang santai melalui tampilan visual yang lembut, animasi interaktif, efek suara, voice-over ringan, serta karakter kecil yang menemani pemain selama bermain.

## Fitur Utama

### 1. Mood Buddy Lek

Game ini memiliki karakter pendamping bernama **Mood Buddy Lek**. Karakter ini berfungsi sebagai teman kecil yang memberikan kalimat santai, lucu, dan menenangkan kepada pemain.

Contoh kalimat yang muncul:

- Sabar dulu lek
- Tarik napas dulu
- Aman lek, pelan-pelan aja
- Minum air dulu
- Jangan chat panjang pas emosi

---

### 2. Tap Emosi

Mode **Tap Emosi** adalah mode permainan di mana pemain menekan bubble emosi yang muncul di layar.

Mode ini cocok untuk pengguna yang sedang:

- Kesal
- Pusing
- Capek
- Sedih
- Bingung
- Overthinking

Saat bubble ditekan, bubble akan pecah dengan efek animasi dan suara ringan.

---

### 3. Breath Tap

Mode **Breath Tap** adalah mode latihan napas ringan di dalam game.

Pemain akan diarahkan untuk mengikuti pola:

- Tarik napas
- Tahan sebentar
- Buang napas pelan-pelan

Mode ini dibuat agar pemain dapat merasa lebih tenang dan fokus.

---

### 4. Pukul Bantal

Mode **Pukul Bantal** adalah mode permainan untuk melampiaskan rasa kesal secara aman melalui bantal virtual.

Pemain cukup menekan bantal di layar. Setiap beberapa tap, akan muncul kata-kata lucu dan provokasi ringan seperti:

- Ayo lek, masa segitu doang?
- Jangan kalah sama bantal
- Overthinking kena hit
- HP jangan jadi korban

Mode ini dibuat agar pengguna bisa melampiaskan emosi secara aman dan lucu.

---

### 5. Fokus 60 Detik

Mode **Fokus 60 Detik** adalah mode permainan yang membantu pemain mengalihkan pikiran.

Pemain harus menekan orb glowing yang muncul dan berpindah posisi selama 60 detik. Mode ini cocok untuk membantu mengurangi overthinking dengan cara mengalihkan fokus sejenak.
